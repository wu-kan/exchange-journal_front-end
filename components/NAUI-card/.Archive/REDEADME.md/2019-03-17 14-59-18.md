#NAUI-card 卡片
使用[ColorUI](https://www.color-ui.com/)封装的卡片组件，组件名:`NAUI-card`,代码块: `NAUIcard`。
ColorUI是一款出色的纯css组件库，可以与任何组件结合，大大加快开发效率!

## 使用方式

在需要使用的页面中引入:

```
import NAUIcard from '@/components/NAUI-card/NAUI-card.vue'
export default {
    components: {NAUIcard}
}
```

在视图中使用

```
<NAUIcard listData="detail"></NAUIcard>
```
